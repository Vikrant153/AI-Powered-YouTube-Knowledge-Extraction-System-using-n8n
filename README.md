# 🎥 AI-Powered YouTube Knowledge Extraction System using n8n

> An AI-powered automation workflow built with **n8n** that extracts knowledge from YouTube videos, analyzes the transcript using Large Language Models (LLMs), and generates structured insights such as summaries, key takeaways, and important concepts.

## 📌 Overview

Watching long educational videos can be time-consuming. This project automates the process of extracting valuable information from YouTube videos by leveraging **n8n** and **AI-powered language models**.

Simply provide a YouTube video URL, and the workflow retrieves the transcript, processes it using an LLM, and generates structured knowledge that is easier to understand and consume.

---

## ✨ Features

- 🎥 Accepts YouTube video URLs
- 📝 Automatically extracts video transcripts
- 🤖 AI-powered transcript analysis
- 📄 Generates concise summaries
- 📌 Extracts key takeaways
- 💡 Identifies important concepts
- ❓ Generates question-answer pairs
- ⚡ Fully automated no-code workflow using n8n
- 🔄 Easily customizable and extensible

## 🚀 How It Works

1. User enters a YouTube video URL.
2. The workflow extracts the transcript.
3. The transcript is cleaned and formatted.
4. The processed content is sent to an AI model.
5. The AI generates:
   - Summary
   - Key Takeaways
   - Important Concepts
   - Question & Answer Pairs
6. The structured knowledge is returned to the user.
